# 🚲 Bike Rental Analysis Dashboard

## 📌 Project Overview
This project analyzes bike rental data to identify key trends in rental demand, rider behavior, and seasonal patterns. An interactive Power BI dashboard was created to present the insights clearly.

## 🛠️ Tools Used
- Power BI
- Data Visualization
- Data Analysis

## 📊 Key Insights
- Total bike rentals and registered riders were analyzed.
- Rental patterns were compared across different time periods.
- Seasonal and weather-related trends affecting bike demand were identified.
- The dashboard provides an interactive view of overall bike rental performance.
